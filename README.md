# AgenTrux Tools for Dify

AgenTrux PubSub tools plugin for Dify.

## Install

In Dify: Plugins → Install from GitHub → `github.com/agentrux/dify-agentrux-tools`

## Tools

- **Publish Event** — publish events to AgenTrux topics
- **Read Events** — read events from topics with pagination
- **Upload File** — upload files via presigned URLs

## Credentials

- **Activation Code** (recommended) — paste `ac_...` code, auto-activates on first use
- **Script ID + Client Secret** — direct credentials if already activated
